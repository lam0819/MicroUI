MicroUI is a lightweight, modern JavaScript utility library that makes DOM manipulation and event handling simple and efficient. It's designed to be fast, small, and easy to use.

The minified version is only 11KB, making it 8.7x smaller than jQuery while providing similar functionality with modern ES6+ features.

MicroUI supports all modern browsers with ES6+ features. It uses native APIs like Web Animations and modern DOM methods for optimal performance.
